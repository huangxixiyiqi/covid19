

# covid19数据分析及可视化


在新冠肺炎疫情全球蔓延的背景下，为帮助公众及相关部门快速掌握疫情动态、辅助决策分析，本项目基于实时疫情数据构建了可视化分析平台。通过整合多源数据、深度清洗加工，结合直观的交互式界面，实现疫情发展趋势、地域分布等关键信息的动态呈现。<br />
前端使用Vue+ElementUI开发，利用Echarts进行数据的图表展示。后端使用Java SPringBoot开发。
